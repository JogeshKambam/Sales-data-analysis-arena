# Sales-data-analysis-arena
## Objective
The objective of this project is to analyze sales data and extract meaningful insights such as total revenue, best-selling product, and sales performance metrics.

## Dataset Overview
- Total records: 100
- Products analyzed: Laptop, Phone, Tablet, Headphones, Monitor
- Dataset contained no missing values after cleaning

## Analysis Steps
1. Loaded the dataset using pandas
2. Explored structure, columns, and data types
3. Cleaned data by handling missing values and removing duplicates
4. Calculated revenue and key metrics
5. Identified best-selling product

## Key Findings
- Laptops generated the highest revenue
- High-priced items contribute more to overall sales
- Dataset quality was good and required minimal cleaning

## Conclusion
The analysis shows that focusing on premium products can significantly increase revenue.
